# Project Summary

This project presents an end-to-end business-focused analysis of Netflix’s global content portfolio using Python, SQL, Excel, Power BI, and Tableau.

The analysis was designed to uncover patterns in:

- Content type distribution
- Country-wise production
- Genre concentration
- Rating segmentation
- Year-wise content growth

The goal of the project is to transform raw streaming platform data into business insights that can support content strategy, audience targeting, and decision-making.

This project demonstrates practical skills in:

- Data cleaning
- Exploratory data analysis
- SQL querying
- Dashboard development
- Business storytelling
- Strategic recommendation building
